# AstroZone.in – UI Design Handover Instructions

This document summarizes all approved creative direction, assets, typography, color palette, and implementation guidance for the AstroZone.in website. Please use this as the source of truth for UI & visual design.

---

## 1. Brand Positioning & Tone

AstroZone.in provides **personalized astrology guidance that becomes more specific over time as users share information**. The UI must communicate:

* Trust
* Warmth
* Guidance
* Premium quality
* A mystical yet modern feel

The tone should feel **reassuring, insightful, elegant, and professional** – not cartoonish or casual.

---

## 2. Core Visual Theme

### Overall direction

* Deep cosmic backgrounds (navy / midnight blue)
* Celestial gold accents
* Clean typography
* Balanced negative space
* Premium visual feel

Think: **cosmic, refined, trustworthy, elegant** – not loud or flashy.

---

## 3. Approved Tagline & Usage

Primary tagline:

> **Astrological Guidance, Made Personal**

This tagline should be displayed prominently on:

* Home page hero banner
* About or mission sections
* Marketing placements where relevant

It should not be overused — strategic placement only.

---

## 4. Logo, Icon & Banner Assets

Use the supplied banner and icons as the basis of the visual system.

Asset pack includes:

### Favicons / Icons (PNG)

* 16×16
* 32×32
* 64×64
* 128×128
* 256×256
* 512×512 (base application icon)

### App Icon

* 512×512 PNG — rounded corners to be handled by platforms where applicable

### Responsive Banners (JPG Optimized)

To be used across screen sizes:

* 2560×900 (Retina / large hero)
* 1920×700 (Desktop primary)
* 1600×600
* 1400×500
* 1200×500 (Standard web header)
* 1080×608 (Mobile wide)
* 1080×1080 (Square / social / mobile)

All banners use aspect-fill and maintain balanced composition.

---

## 5. Web Font Pairing

Recommended font families:

### Headings & Brand Titles

**Playfair Display** or **Cormorant Garamond**

Usage characteristics:

* Serif
* Elegant
* Premium editorial style

### Body Text

**Poppins** or **Source Sans 3**

Usage characteristics:

* Simple
* Clean
* Highly readable

### Optional Accent Font

**Cinzel** – for small decorative labels / astrology headings only (minimal use).

## 6. Brand Color Palette

### Primary

* **Deep Cosmos Blue** — `#0B1437`
* **Royal Midnight** — `#131C4B`

### Accent / Highlight

* **Celestial Gold** — `#D4AF37`
* **Soft Moonlight Cream** — `#F2E8D5`

### Supporting / Secondary

* **Star Glow** — `#FFDCA8`
* **Nebula Violet** — `#6A5FAE`

### Neutral Text Colors

* Primary text — near-white / cream
* Avoid harsh pure white if possible

---

## 7. Layout & UI Style Guidance

### Hero Banner

* Banner should use one of the supplied responsive sizes
* Title: **AstroZone.in** (serif font)
* Subtitle/tagline displayed below
* Adequate breathing space (do not crowd the center)

### Section Style

* Clean section breaks
* Strong typography hierarchy
* Minimal UI chrome

### Button Style

* Rounded or subtly rounded
* Gold or soft cream against dark backgrounds

Hover state:

* Slight glow OR lighter gold

Avoid:

* Neon colors
* Harsh contrasts
* Excess gradients

---

## 8. Iconography & Symbol Language

Icons should:

* Be consistent in line weight & style
* Avoid cartoon characteristics
* Feel ceremonial / classical where possible

When symbolizing zodiac elements, maintain **respectful and refined artistic directions**.

---

## 9. Accessibility Guidelines

Ensure:

* Minimum contrast ratio 4.5:1
* Legible font size (minimum 16px body)
* Sufficient line spacing
* Avoid long line-lengths

Dark mode friendliness should be maintained.

---

## 10. Performance Requirements

* Use web-optimized image versions where possible
* Prefer JPG or WebP for banners
* Lazy-load below-the-fold media

Goal: **fast-loading, visually rich experience**.

---

## 11. Future-State UI Considerations

Since the platform personalizes guidance over time, the UI should visually support a user journey, including:

* Phased onboarding questions
* A sense of progression / clarity increasing
* Trust cues

Examples:

* Progress indicators
* Gentle animations
* Subtle emphasis on discovery & accuracy

---

## 12. Deliverables Expected from UI Designer

You will be expected to provide:

* Final responsive UI layouts (desktop + mobile)
* Component style system / design system
* Final favicon & app icon placements
* Exported optimized graphics
* CSS/Dev-ready specs OR handoff via Figma/Zeplin/etc.

---

## 13. Collaboration Notes

Please preserve consistency with the supplied brand direction. If in doubt:

* Prioritize clarity
* Avoid over-decoration
* Preserve a premium experience
